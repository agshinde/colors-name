🎨 Colors Name

A simple educational web project that helps users learn color names in Hindi through visual examples.

📖 About the Project

This project displays different colors along with their Hindi names and corresponding images. It is designed as a beginner-friendly frontend project using HTML and CSS.

✨ Features
Display color names in Hindi
Visual representation of each color using images
Clean and simple user interface
Beginner-friendly project structure
🛠️ Technologies Used
HTML5
CSS3
